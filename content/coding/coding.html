JSON.parse() Implementation

View source code

A recursive descent parsing technique is used to implement JavaScript's JSON parser. JSON.stringify() only stringifies: objects (including arrays), numbers, strings, booleans and null. NOT functions, undefined, or NaN.

Type in any JSON-stringified JavaScript value and click Parse!

Wikipedia API Web Crawler

View source code

This search bar crawls randomly through Wikipedia pages:

  1. Type one or more keywords and click Search. A Wikipedia page title that matches it will be displayed with a description
  2. Within this Wikipedia page's content, a randomly chosen word will be searched and another matching Wikipedia page will be found and displayed, with its description.
  3. So on and so forth, displaying 50 connected Wikipedia pages! (Any problems just refresh the page or try other keywords.)

Compare the first and last outputs!
